## Prerequisites
***
- npm = 8.19.2
- node = v18.12.1

## 설치법
***
```sh
npm install -g eslint prettier

npm install
```

## 로컬 실행법 / 배포 소스 빌드법
***
```sh
npm run start # 로컬에서 실행
 
npm run build-dev # dev 환경용 build 파일 생성

npm run build-stage # stage 환경용 build 파일 생성 

npm run build-prod # prod 환경용 build 파일 생성
```
